# DGS (D'Genius Solutions) Project Guidelines & Design System

## Core Design Principles & References
- **Aesthetic Direction:** Ultra-premium, Awwwards/Linear/Raycast/Runway/Stripe-level creative dark-mode interface.
- **Background & Visuals:** Sleek obsidian dark background (`#050508` or `#08090a`) with volumetric organic fluid gradients, soft glowing ambient orbs, and rich organic 3D particle landscapes. NO harsh low-poly or basic wireframe geometric shapes.
- **Controlled 3D Strategy:** Use 3D (Three.js WebGL / CSS 3D / Spline) ONLY in selected key visual areas so the site remains ultra-fast, mobile-friendly, SEO-ready, and conversion-focused.
- **Typography:** Use **Manrope** globally across all elements.

## Priority Services Focus
1. **AI Video Production** (AI Avatars, Hyper-realistic Visuals, TVCs, Viral Shorts)
2. **SEO, AEO, GEO & LLM Optimization** (Google AI Overview, Perplexity, ChatGPT Search ranking)
3. **Website Development** (High-Performance Next.js, WebGL, Interactive Web Apps)
4. **Social Media Marketing** (Community Growth, Brand Engagement, Viral Campaigns)
5. **Performance Marketing** (High-ROAS Paid Search, Meta Ads, Funnel Optimization)
6. **Branding & Content Creation** (Brand Identity, Copywriting, Creative Asset Design)

## Design Inspiration Pool
- **Awwwards / Behance / Dribbble / Godly / Land-book / SiteInspire:** High-end agency layouts, visual storytelling, dark UI micro-interactions.
- **Linear / Raycast / Runway / Stripe:** Refined glassmorphism, pill badges, glowing gradient borders, crisp contrast, refined motion.
- **GSAP ScrollTrigger & Three.js:** Smooth scroll-driven typography movement, section transitions, controlled WebGL fluid/particle waves.

## Technical & Performance Rules
- Preserve 100% of semantic HTML, JSON-LD Schema (Organization, ProfessionalService, FAQPage, WebSite), and SEO metadata for AEO/GEO/LLM indexing.
- Prioritize Core Web Vitals, fast mobile rendering, high FPS, and seamless responsive scaling from 4K ultrawide down to 320px mobile screens.
